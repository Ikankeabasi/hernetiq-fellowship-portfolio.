# Week 1 – AI Architecture Diagram

This project contains my 4-layer AI architecture diagram for ChatGPT, created as part of the HerNetIQ AI Security Fellowship Week 1 assignment.

I mapped the Data, Model, API, and Infrastructure layers and identified one security risk for each layer from a defensive security perspective.

While building this diagram, I learned that understanding AI security starts with understanding how data flows through the entire AI system, not just the AI model itself.
