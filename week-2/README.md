# Week 2

## Session 3 – Defensive Python

Reviewed a vulnerable Python configuration file and identified multiple security issues, including hardcoded API keys, wildcard cloud permissions, missing authentication, and exposed secrets. Documented all findings in a structured security report.

## Session 4 – JSON Log Analysis

Analysed AWS CloudTrail JSON logs using the IOC 5Ws framework to investigate a privilege escalation attack. Traced how a stolen API key was used to obtain administrator access and documented the investigation findings.
